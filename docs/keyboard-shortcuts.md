# Keyboard Shortcuts
## Google Hangouts
```sh
# Toggles microphone
cmd+d

# Toggles camera
cmd+e
```

## OSX iTerm

### Navigation
```sh
# Clears line
ctrl+l

# Deletes last word
ctrl+w

# Deletes words before cursor
ctrl+u

# Deletes words after cursor
ctrl+k

# Jumps to start of line
ctrl+a

# Jumps to end of line
ctrl+e
```

### Transformation
```sh
# Uppercase word
opt+u

# Lowercase word
opt+l
```

### Window
```sh
# Toggles terminal transparency
cmd+u

# Opens preferences
cmd+,
```
